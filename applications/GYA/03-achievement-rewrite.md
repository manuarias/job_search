## Patterns & Insights

1. **El perfil real es fuerte para ambas JDs** — el problema es que está narrado como management, no como desarrollo.
2. **Python** aparece solo en EXO CORP (2017-2018) — es reciente pero hay que destacarlo más.
3. **SQL Server es el gap real** — no aparece en el CV ni siquiera como experiencia pasada. Hay que decidir si:
   - a) El usuario tiene experiencia con SQL Server no listada
   - b) Framing con MySQL como "bases de datos relacionales incluido SQL Server"
4. **Frontend** — no hay experiencia real. Las JDs piden "sistemas web" sin especificar stack. Podemos focusing en "sistemas web" como APIs + consumo de APIs.
5. **Dashboards** — no hay experiencia explícita. Preguntar al usuario.

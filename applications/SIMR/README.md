## Key Decisions Made

1. **Keep the Project Leader role, but reframe it.** Employment gaps are worse than leadership history. The role is titled "Project Leader (Hands-on Technical Lead)" and bullets emphasize architecture, debugging engines, and API contracts — not roadmaps and stakeholders.
2. **Do NOT add MySQL.** The source CV mentions CouchDB and MongoDB but never MySQL. Adding it would be dishonest. It is marked as [NEEDS DATA] for the user to verify.
3. **Do NOT add Angular.** It is a nice-to-have and the candidate does not have it.
4. **Do NOT claim SOLID or unit/integration tests** without user confirmation. These are must-haves in the JD but absent from the source CV.
5. **Use English narrative throughout.** The JD is in English and the output language is English. All technical keywords remain in English.

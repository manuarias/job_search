Salesforce Developer Junior

## Sobre Vantegrate

En Vantegrate somos un equipo de profesionales que transforma desafíos complejos en soluciones que cambian la realidad de las empresas, sus empleados y sus clientes. Nos mueve la pasión por la tecnología y el deseo de generar impacto en el mundo empresarial, construyendo experiencias únicas para clientes y usuarios finales.

Si te motiva trabajar en un ambiente dinámico, colaborativo y orientado a resolver problemas reales, este es el lugar para vos.

## De qué se trata el puesto

Buscamos un Salesforce Developer Junior comprometido y entusiasta para sumarse a proyectos transformadores. Vas a ser parte fundamental en la personalización y el desarrollo de aplicaciones sobre la plataforma Salesforce, trabajando junto a equipos de primer nivel para potenciar la experiencia del cliente y optimizar procesos internos mediante soluciones tecnológicas de alto impacto.

## Objetivo del rol

Desarrollar soluciones que integren Salesforce con sistemas externos mediante n8n, APIs y componentes de inteligencia artificial. Buscamos un perfil técnico de nivel intermedio, con capacidad para ejecutar de manera autónoma y aprender rápido las herramientas del ecosistema Vantegrate.

## Principales responsabilidades

### Desarrollo y personalización
- Diseñar, desarrollar e implementar soluciones personalizadas en Salesforce utilizando Apex, Flow y Lightning Web Components.
- Integrar módulos y componentes que permitan una interacción fluida con herramientas de inteligencia artificial.

### Automatización de procesos
- Diseñar y mantener flujos de trabajo y automatizaciones que incrementen la eficiencia operativa y reduzcan los errores manuales.
- Identificar oportunidades para automatizar tareas repetitivas y optimizar recursos.

### Documentación y mejora continua
- Elaborar y mantener actualizada la documentación técnica y operativa, asegurando que los procesos estén bien definidos y sean replicables.
- Proponer mejoras y ajustes basados en feedback de usuarios y métricas de rendimiento.

### Trabajo en equipo y colaboración
- Participar activamente en reuniones interdisciplinarias para comprender las necesidades del negocio y traducirlas en soluciones técnicas robustas.
- Aportar ideas y sugerencias en un ambiente donde la innovación y el feedback son altamente valorados.

## Conocimientos técnicos requeridos

### Salesforce
- Conocimiento de Apex (clases, triggers, tests unitarios).
- Experiencia con Flows (Screen Flows, Record-Triggered Flows, Subflows).
- Manejo de APIs REST de Salesforce, tanto para consumir como para exponer endpoints custom.
- Nociones de Lightning Web Components (LWC). No pedimos expertise, pero sí que puedas leer y modificar componentes existentes.
- Comprensión del modelo de datos de Salesforce: objetos estándar, custom objects, relaciones (lookup, master-detail) y configuración de permisos.

### n8n y automatización
- Experiencia armando workflows en n8n o herramientas equivalentes. Make o Zapier cuentan, n8n es un plus fuerte.
- Capacidad de configurar webhooks, nodos HTTP Request y transformaciones de datos (JSON, expresiones).
- Experiencia o disposición para integrar n8n con Salesforce y APIs de terceros.
- Manejo básico de errores dentro de workflows (reintentos, rutas de error).

### Agentes de IA (nociones)
- Entender qué es un agente de IA, cómo se estructura un prompt y cómo interactúa un agente con herramientas externas (tools y function calling).
- No se requiere construir agentes desde cero, pero sí poder configurar y ajustar agentes existentes, entender flujos conversacionales y colaborar en el diseño de soluciones con IA.
- Familiaridad con conceptos como prompt engineering, contexto, temperatura y modelos de lenguaje (LLMs).

### APIs y programación orientada a objetos
- Comprensión clara de APIs REST: verbos HTTP (GET, POST, PUT, DELETE), autenticación (OAuth, API keys), headers y body en JSON.
- Conceptos sólidos de POO: clases, objetos, herencia, encapsulamiento y polimorfismo, aplicados en Apex, JavaScript o Python.
- Capacidad de leer documentación técnica de APIs y traducirla a implementación.

### Bases de datos
- Modelado de datos relacional: tablas, relaciones, claves primarias y foráneas, normalización básica.
- Manejo de SOQL (Salesforce) o SQL estándar.
- Entender la diferencia entre bases de datos relacionales y el modelo de datos de Salesforce.

### Nice to have (no excluyente)
- Experiencia con Git y flujos de trabajo colaborativo (branches, pull requests).
- JavaScript moderno (ES6+, async/await, fetch).
- Experiencia con Agentforce u otras plataformas de agentes de IA.
- Nociones de Python para scripts de integración o automatización.
- Inglés técnico para lectura de documentación.
- Experiencia en migración de datos y pruebas de calidad en Salesforce.
- Conocimientos en seguridad y cumplimiento normativo dentro de la plataforma.
- Familiaridad con herramientas y tecnologías de inteligencia artificial que aporten valor agregado a los proyectos.

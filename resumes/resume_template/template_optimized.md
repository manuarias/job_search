## CÓMO USAR ESTE TEMPLATE

### Paso 1: Encontrá la JD
Antes de tocar este archivo, conseguí la Job Description del rol al que querés aplicar.

### Paso 2: Identificá los Must-Haves
Marcá con resaltador:
- Título del rol (exactamente como lo escriben ellos)
- Las 5-10 skills/tools que repiten más de una vez
- Los verbos de acción que usan ("drive", "own", "orchestrate", "translate")

### Paso 3: Adaptá el contenido
| Sección | Qué cambiar | Qué mantener |
|---------|-------------|--------------|
| **Header** | Títulos según el rol (ej: "Technical Program Manager") | Formato y contacto |
| **Summary** | Bullets según los 3 mensajes clave del JD | Estructura de 3 bullets |
| **Core Competencies** | Las 4 áreas según los Must-Haves del JD | Formato de 4 bullets |
| **Core Skills** | Categorías y skills según el rol | Formato de lista |
| **Experience** | Verbos y framing según el rol (TPM vs Engineer vs Manager) | Fórmula métrica + resultado |
| **Education** | Ordenar por relevancia al rol | Formato |

### Paso 4: Verificá keywords
Corré este checklist antes de enviar:

- [ ] ¿El título del rol aparece en el header o summary?
- [ ] ¿Las top 5 keywords de la JD están en Core Competencies?
- [ ] ¿Cada bullet de experiencia tiene al menos una métrica?
- [ ] ¿No hay párrafos de más de 3 líneas?
- [ ] ¿Las fechas están en formato "Jan 2023"?
- [ ] ¿No hay code blocks (```), tablas, ni caracteres especiales raros?
- [ ] ¿El CV entra en 1-2 páginas?

### Paso 5: Scoreá antes de enviar
Usá esta rubrica rápida:

| Pregunta | Sí / No |

## REGLAS DE ORO

1. **Un CV = una JD.** Nunca envíes el mismo CV a dos roles distintos sin adaptarlo.
2. **El lenguaje del CV debe ser el lenguaje de la JD.** Si ellos dicen "stakeholder alignment", vos decís "stakeholder alignment". No "trabajé con gente de negocio".
3. **Métricas > Responsabilidades.** "Reduje incidentes" es débil. "Reduje incidentes un 70%" es fuerte.
4. **Los primeros 7 segundos deciden todo.** Si el recruiter no ve valor en el header + summary + Core Competencies, no scrollea.
5. **No mientas.** Si no usaste "Linear", no lo pongas. Pero si usaste Notion/ClickUp/Trello, podés decir "experience with workflow tools including Jira and Linear-like platforms".

# Desarrollador Full Stack - FakeCorp Latam

## Acerca del empleo
Desarrollador Full Stack Semi-Senior

Modalidad 100% remota [solo Argentina]

## Sobre FakeCorp Latam

En FakeCorp Latam somos un equipo de profesionales que transforma desafíos complejos en soluciones que cambian la realidad de las empresas.

## De qué se trata el puesto

Buscamos un Desarrollador Full Stack comprometido y entusiasta para sumarse a proyectos transformadores. Vas a ser parte fundamental en el desarrollo de aplicaciones, trabajando junto a equipos de primer nivel.

## Objetivo del rol

Desarrollar soluciones que integren sistemas mediante **n8n**, **APIs RESTful** y componentes de automatización. Buscamos un perfil técnico de nivel intermedio, con capacidad para ejecutar de manera autónoma y aprender rápido.

## Principales responsabilidades

### Desarrollo y personalización
- Diseñar, desarrollar e implementar soluciones personalizadas utilizando **JavaScript**, **Python** y **SQL**.
- Integrar módulos y componentes que permitan una interacción fluida con herramientas de automatización.

### Automatización de procesos
- Diseñar y mantener flujos de trabajo en **n8n** y automatizaciones que incrementen la eficiencia operativa.
- Identificar oportunidades para automatizar tareas repetitivas y optimizar recursos.

### Trabajo en equipo y colaboración
- Participar activamente en reuniones interdisciplinarias para comprender las necesidades del negocio.
- Aportar ideas y sugerencias en un ambiente donde la innovación y el feedback son altamente valorados.

## Conocimientos técnicos requeridos

### Desarrollo
- Conocimiento de **JavaScript** (ES6+, async/await, fetch).
- Experiencia con **Python** para scripts de integración o automatización.
- Manejo de **APIs REST**: verbos HTTP (GET, POST, PUT, DELETE), autenticación (OAuth, API keys), headers y body en JSON.
- **SQL** estándar para modelado de datos relacional.

### n8n y automatización
- Experiencia armando workflows en **n8n** o herramientas equivalentes.
- Capacidad de configurar webhooks, nodos HTTP Request y transformaciones de datos (JSON, expresiones).

### Control de versiones
- Experiencia con **Git** y flujos de trabajo colaborativo (branches, pull requests).

### Metodologías
- Experiencia trabajando con **Scrum** y metodologías ágiles.

### Nice to have (no excluyente)
- Nociones de Python para scripts de integración o automatización.
- Inglés técnico para lectura de documentación.

## Nivel esperado

Perfil semi-senior. No esperamos que sepas todo desde el día uno, pero sí que puedas tomar un requerimiento, investigar lo necesario y resolverlo con autonomía razonable.
